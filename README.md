# go-microservices

## Project: User & Notification Microservices
Create two microservices that communicate via gRPC, with message queue integration.

Incremental learning using Claude Code in the terminal as a Guide, all code is researched and written by hand to ensure maximum practice, then the agent gives corrections and next steps.

Project Overview

- Phase 1 Project Setup: Go modules, project structure
- Phase 2 Basic HTTP Server: net/http, then Gin
- Phase 3 Task CRUD (in-memory): Structs, slices, handlers
- Phase 4 Database Integration: database/sql and Postgres driver
- Phase 5 User Auth: JWT, password hashing
- Phase 6 Config Management: Environment variables
- Phase 7 Docker: Containerization
- Phase 8 Validation & Errors: Structured responses
